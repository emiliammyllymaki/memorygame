# Kitty Memory 🐱

A cat-themed memory card game built with Flutter and Flame.

## Description

Kitty Memory is a card matching game where the player flips cards to find matching cat themed emoji pairs. The game features three levels of increasing difficulty. Progress is saved between sessions and each new level unlocks after completing the previous one.

## How to Play

1. Tap **Play!** on the start screen.
2. Select a level. 
3. Tap a card to flip it and reveal the emoji.
4. Tap a second card, if the emojis match, both cards stay face up.
5. If they don't match, both cards flip back after a short delay.
6. Find all pairs to complete the level.
7. The fewer moves you use, the better your cat rating!
8. The next level unlocks after completing the current one.

## Levels

- 🐱 **Level 1 — Kitten**: 6 pairs (easy)
- 🐈 **Level 2 — Cat**: 10 pairs (medium)
- 👑 **Level 3 — Cat Queen**: 15 pairs (hard)
